      v1.0.0
      2026-05-31

      🚀 專案正式誕生！

      Language Journey 核心功能部署完成，正式上線 GitHub Pages！

      • Dashboard 儀表板：快速檢視今日打卡狀態、連續天數與累積時間。
      • 每日打卡系統：記錄每日學習內容與時間。
      • GitHub 風格熱力圖：動態生成過去 365 天的學習進度矩陣。
      • 錯題紀錄與隨機複習：卡片式錯題管理，搭配單字卡閃卡複習模式。
      • 統計報告：串接 Chart.js 圖表，自動分析近一週學習時間與月達成率。
      • LocalStorage 永久存儲：免帳號、免後端，資料安全儲存於個人瀏覽器，支援 JSON 備份匯出。
      • 雙色主題：支援 Notion 感淺色模式與極簡深色模式。
